## Overview

QtPy5を使ってお手軽にGUI操作でmastodonに連投を行えるツール。
※使用の際は節度を守って使用して下さい

## Requirement

```
pip install PyQt5
pip install SIP
pip install mastodon.py
```

## Install

```
git clone https://github.com/Haritaso/rentoukun.git
cd rentoukun
python main.py
```
## Usage

左側を上から順番に入力後、右側のツールが使用可能になります。
knzk.meの部分は自由に変更が可能です。
